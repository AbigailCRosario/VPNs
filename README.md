# VPNs
Practica de configuracion de diferentes tipos de VPN, como site-to-site, hub and spoke y client-to-site. 

# 🌐 Implementación y Análisis de Túneles VPN (IPsec)

¡Bienvenido/a a mi repositorio de prácticas de redes y seguridad! 

Este proyecto documenta la configuración, despliegue y verificación de 9 topologías de VPN diferentes. Las primeras fases del proyecto se centran en una comparativa directa entre las implementaciones de **IKEv1** e **IKEv2** para evaluar sus diferencias en sintaxis, negociación de fases y seguridad.

Los laboratorios fueron emulados y validados utilizando entornos de virtualización de red como **GNS3** y **PNETLab**.
